# -Furniture-Ecommerce-Product-Data-
# README.md: 競合製品データ分析による市場競争力強化

## プロジェクト概要

本プロジェクトは、`competitors_raw_data.csv`データセットに含まれる競合製品の情報を詳細に分析し、自社製品の市場競争力強化を目的とした戦略的インサイトを導出することを目的としています。価格、評価、機能、カテゴリといった多角的な視点から競合製品を評価し、KGI（Key Goal Indicator）である「自社製品の市場競争力強化」に貢献する具体的なアクションプランを提案します。

## データ概要

### データセット: `competitors_raw_data.csv`

このデータセットは、競合製品に関する以下の8つの列を持つ247行のデータで構成されています。

*   **`Title`**: 製品のタイトル
*   **`Price`**: 製品の価格
*   **`Category_path`**: 製品が属するカテゴリパス（階層構造）
*   **`Rating`**: 顧客による製品の平均評価スコア
*   **`Qty_califications`**: 製品の評価数
*   **`Features_JSON_format`**: 製品の機能リスト（JSON形式）
*   **`Image_Url`**: 製品画像のURL
*   **`Item_Url`**: 製品詳細ページのURL

### データ理解の要点:

*   **欠損値**: `Rating`, `Qty_califications`, `Features_JSON_format`に欠損値が存在し、これらはデータクリーニングフェーズで適切に処理されました。
    *   `Rating`と`Qty_califications`は中央値で補完。
    *   `Features_JSON_format`は空のJSONオブジェクト文字列で補完。
*   **外れ値**: `Price`, `Rating`, `Qty_califications`列においてIQR法を用いた外れ値が検出されました。これらはビジネス上重要な意味を持つ可能性があり、分析では維持されました。
*   **異常値**: 業務ロジックに基づく異常値（例: 価格が0以下）は検出されませんでした。
*   **データ品質スコア**: データ品質スコアは96.96点と非常に高く、分析結果の信頼性が高いことを示唆しています。

## 分析手法

本分析は、以下の主要なステップで構成されています。

1.  **ビジネス理解**: KGI（Key Goal Indicator）、KPI（Key Performance Indicator）、およびKPIツリーを定義し、分析対象を明確化しました。
    *   **KGI**: 自社製品の市場競争力強化
    *   **KPI**: 競合製品の平均価格、平均評価スコア、平均評価数、主要カテゴリにおける競合製品数、特定機能を持つ競合製品の割合
    *   **KPIツリー**: KGIとKPIの因果関係をmermaid形式で視覚化し、相互関係を整理しました。

2.  **データクレンジングと前処理**: 欠損値の処理、データ型変換（`Features_JSON_format`をJSONオブジェクトへ）、外れ値・異常値の検出を行いました。

3.  **探索的データ分析 (EDA)**: `Price`, `Rating`, `Qty_califications`, `Category_path`, `Features_JSON_format`の各列について、記述統計量の算出、ヒストグラムや箱ひげ図による視覚化、上位カテゴリ・機能の分析を実施しました。

4.  **仮説検証**: EDAで生成された3つの仮説を統計的に検証しました。
    *   **仮説 1**: 価格と評価スコアの関係性
    *   **仮説 2**: 評価数と人気機能の関係性
    *   **仮説 3**: カテゴリごとの市場特性

5.  **戦略策定**: 検証された仮説と効果量に基づき、具体的なアクションプラン（価格設定、機能開発、品質向上、市場選定）を策定しました。

6.  **Impact x Effort分析**: 策定した戦略に対してImpact（影響度）とEffort（労力）の評価基準を適用し、実行優先順位を決定しました。

## 主要インサイト

### EDAからの洞察:

*   **価格分布**: 低価格帯に多くの製品が集中する一方で、高価格帯にもニッチな製品が存在。市場は価格多様性が高い。
*   **評価スコア分布**: 全体的に高評価の製品が多く、市場の顧客満足度は高い水準にある。低評価の製品も少数存在するが、多くの製品が高品質を提供している。
*   **評価数分布**: 多くの製品は評価数が少ないが、一部の製品は非常に多くの評価を集め、高い人気と認知度を持つ。
*   **カテゴリ多様性**: データセットは非常に細分化されたカテゴリパスを持ち、特定のカテゴリに製品が集中する傾向は見られない。市場の細分化が進んでいることを示唆。
*   **機能トレンド**: 「Material」「Descripcion_0」「Anchura exterior」といった基本的な構成要素や情報に関する機能が最も頻繁に出現する。
*   **カテゴリ別比較**: 上位カテゴリ間で価格、評価スコア、評価数に傾向の違いが見られ、カテゴリごとに異なる市場特性や競争環境が存在。

### 仮説検証結果からのインサイト:

*   **仮説1: 価格と評価**: 「高価格帯の競合製品が低価格帯の製品より平均評価スコアが統計的に有意に高い」という仮説は支持されませんでした（P値: 0.171, Cohen's d: -0.176）。これは、単に価格を高くするだけでは評価に直結しないことを示唆しています。価格以外の要因（品質、機能、ブランドなど）が評価に大きく影響している可能性があります。
*   **仮説2: 評価数と人気機能**: 「人気製品群は特定の機能の搭載割合が有意に高い」という仮説は、**Material**, **Altura total**, **Anchura interior**, **Número de baldas**, **Grueso de laterales y baldas**, **Fondo estantería** の機能において統計的に有意な差が見られましたが、Cramer's Vによる効果量は小さい傾向にありました。これは、これらの機能が人気獲得に寄与する可能性はあるものの、決定的な要因ではないことを示唆します。製品の人気は複合的な要因で決定されると考えられます。
*   **仮説3: カテゴリ別市場特性**: 選定された2つのカテゴリ（'Cama 90 con cajón y colchón'と'Estructura de cama 105'）間では、**Price** (P値: 0.005, Cohen's d: 17.102) と **Rating** (P値: 0.042, Cohen's d: 15.000) において統計的に有意かつ非常に大きな差が確認されました。しかし、**Qty_califications** (P値: 0.223, Cohen's d: 1.800) では有意な差は見られませんでした。これは、カテゴリ間で価格設定や顧客評価の期待値に明確な違いがある一方で、評価数についてはこれらの特定のカテゴリ間では顕著な差がないことを示唆します。

## 施策

検証結果と効果量の分析に基づき、以下の4つの具体的な行動戦略を策定しました。

1.  **価格設定と価値訴求戦略**: 顧客が納得する「価値」に基づいた価格設定を行い、ターゲット市場セグメントに合わせて最適化。高価格帯製品では品質・独自機能・顧客体験を明確に訴求する。
2.  **製品機能の優先順位付けと開発戦略**: 人気製品に共通する基本機能を自社製品に標準装備し、競合より優れた形で提供。同時に、市場の差別化要因となるニッチな機能や革新的な機能の探索・開発を継続する。
3.  **品質向上と顧客体験戦略**: 全価格帯の製品で一定水準以上の品質を確保し、顧客レビュー分析を通じて品質改善に活かす。特に高価格帯製品では、徹底した品質管理と優れた顧客サポートで、高価格を正当化する顧客体験を創出する。
4.  **ターゲット市場とカテゴリ選定戦略**: 競合の「Category_path」データを深く分析し、カテゴリごとの市場特性を把握することで、自社が強みを活かせるニッチ市場や成長市場を特定し、ターゲットカテゴリに特化した戦略を策定する。

## ROI評価と限界

### ROI分析のためのデータ利用可能性:

`competitors_raw_data.csv`データセットには、製品の原価、販売数量、総収益、マーケティング費用といったROI（Return on Investment）分析に不可欠なデータが含まれていません。したがって、このデータセット単独で包括的なROI分析を行うことはできません。

### 代替パフォーマンス指標:

既存のデータから間接的に経済的価値や市場の成功度合いを推測するため、以下の代替指標を提案します。

*   **相対価格競争力**: 製品価格を同一カテゴリ内の競合製品の平均価格と比較。
*   **評価効率**: `Rating × log(Qty_califications + 1)`で評価の質と量を複合的に評価。
*   **人気度指数**: `Qty_califications`を基に市場での相対的な人気度を測る。
*   **プレミアム度合い**: 高価格帯で高評価を獲得している製品の度合いを示す。

### 分析の限界:

*   **データ範囲の制約**: 分析は提供された競合製品データに限定されており、自社製品のデータや市場シェア、財務情報など広範なデータは含まれていません。
*   **サンプルサイズの制約**: カテゴリ別の比較分析では、各カテゴリの製品数が少ないため、統計的検定の信頼性や一般化に限界があります。特に'Qty_califications'の比較では、p値の信頼性が低い可能性が指摘されました。
*   **機能情報の粒度**: `Features_JSON_format`はJSON形式ですが、その内容の均一性や構造にばらつきがあり、詳細な機能比較には追加の前処理が必要となる可能性があります。

## Future Work / Next Actions

## Impact x Effort 分析

### Impact x Effort マトリックス: 行動戦略の優先順位付け

### 戦略の実行優先順位

Impact x Effort分析に基づき、以下の原則で行動戦略の実行優先順位を決定しました。
「Impactが高いほど優先度が高く、同じImpactレベルであればEffortが低いほど優先度が高い」

1.  **品質向上と顧客体験戦略**
    *   **Impact**: High
    *   **Effort**: Medium
    *   **優先順位の理由**: 顧客ロイヤリティとブランドイメージに直接影響し、長期的な市場競争力強化に不可欠であるため、高いインパクトを持ちます。品質管理プロセスの強化や顧客サポート体制の改善には労力がかかるものの、大規模な製品開発に比べれば中程度であるため、早期に着手し、その効果を最大化すべきです。

2.  **製品機能の優先順位付けと開発戦略**
    *   **Impact**: High
    *   **Effort**: High
    *   **優先順位の理由**: 基本機能の改善は顧客満足度向上に直結し、差別化機能は新たな顧客層獲得に繋がり、KGIへの影響は非常に高いです。しかし、機能開発には研究開発、設計、製造工程の変更など多大な時間、予算、技術リソースが必要となるため、最も高い労力を伴います。中長期的な視点での計画とリソース配分が必要であり、初期段階での市場と顧客のニーズのさらなる詳細な調査とプロトタイプ開発から始めることが推奨されます。

3.  **ターゲット市場とカテゴリ選定戦略**
    *   **Impact**: Medium
    *   **Effort**: Low
    *   **優先順位の理由**: 適切なターゲット市場とカテゴリの選定は、リソースの効率的な配分と効果的な戦略策定を可能にし、KGIに中程度の貢献をします。主にデータ分析と戦略立案が中心であり、即座に大きな投資を伴わないため労力は低いです。市場理解を深め、他の戦略の精度を高めるために非常に重要であるため、即座に実行を開始すべきです。

4.  **価格設定と価値訴求戦略**
    *   **Impact**: Medium
    *   **Effort**: Medium
    *   **優先順位の理由**: 価格設定の調整は比較的迅速に可能ですが、価値訴求のためにはマーケティング戦略の見直しやブランドメッセージの強化が必要です。市場セグメント別の最適化にはある程度の市場分析と実行期間を要するため、中程度のインパクトと労力と評価されます。品質向上や市場理解が進んだ段階で、より洗練された形で実施することでその効果を高めることができます。

### 全体的なビジネスへの示唆と推奨されるアプローチ

今回の優先順位付けは、限られたリソースの中で最大の効果を得るための指針となります。特に、「**品質向上と顧客体験戦略**」は高いインパクトを持ちながら労力が中程度であるため、早期に着手し、その効果を最大化すべきです。これと並行して、労力が低い「**ターゲット市場とカテゴリ選定戦略**」は、市場理解を深め、他の戦略の精度を高めるために非常に重要であり、即座に実行を開始すべきです。「**製品機能の優先順位付けと開発戦略**」は高いインパクトが期待されるものの、最も高い労力を伴うため、中長期的な視点での計画とリソース配分が必要です。初期段階での市場と顧客のニーズのさらなる詳細な調査とプロトタイプ開発から始めることが推奨されます。「**価格設定と価値訴求戦略**」は中程度のインパクトと労力であり、品質向上や市場理解が進んだ段階で、より洗練された形で実施することでその効果を高めることができます。

これらの戦略を段階的に、かつ相互に関連付けながら実行することで、持続的な市場競争力の強化が期待できます。
